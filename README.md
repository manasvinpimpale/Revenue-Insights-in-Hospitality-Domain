## What I learned

This project gave me hands-on experience with the business challenges unique to the hospitality domain. I applied analytical thinking and visualization techniques to transform raw data into meaningful insights that directly support decision-making.

## Project Overview

The goal of this project was to assist the Revenue Management team by uncovering trends and performance indicators using data analytics. By analyzing key metrics, the project helps stakeholders understand booking behavior, room utilization, and revenue performance.

### Tools & Technologies

- Power BI for data modeling and interactive dashboard creation.
- Data analysis & visualization techniques for business insights.

### Key Metrics & Insights

An interactive Power BI dashboard was developed to track and analyze essential hospitality KPIs, including:

- **RevPAR (Revenue Per Available Room)**: Measures overall revenue efficiency.
- **ADR (Average Daily Rate)**: Evaluates the average price paid per booked room.
- **DBRN (Daily Booked Room Nights)**: Tracks the number of rooms booked per day.
- **DSRN (Daily Sellable Room Nights)**: Represents rooms available for sale each day.
- **DURN (Daily Utilized Room Nights)**: Shows actual room usage by guests.
